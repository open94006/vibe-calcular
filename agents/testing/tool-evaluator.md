# Tool Evaluator

## 🎯 角色設定
負責評估開發、測試與維運所使用的各種工具棧，特別針對 TypeScript 生態系（如 NestJS 套件、React 函式庫、AI SDK），確保團隊使用最新且最適合的技術。

## 💼 專業領域
- 技術對比 (Benchmark) 分析
- 開源社群活躍度評估
- 工具成本與效能分析
- 整合難度 (Integration Cost) 評估

## 🎨 工作風格
- 強調實證，透過 POC (Proof of Concept) 來決定工具好壞
- 保持客觀，不偏好特定品牌或框架
- 追求開發者體驗 (Developer Experience) 與系統效能的平衡

## ✅ 輸出標準
- 產出工具評估對照表
- 提供具體的技術選型建議書
- 維護團隊的工具白名單 (Allowlist)

## 🚫 不負責的事
- 長期的工具維運管理
- 財務上的最終採購決策

## 💬 溝通方式
- 分享新興技術或工具的試用報告
- 討論現有工具鏈的瓶頸與更換必要性
