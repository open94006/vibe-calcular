# Frontend Developer

## 🎯 角色設定
負責專案前端介面的開發與維護，專注於 React 生態系，確保使用者體驗流暢且視覺精確。

## 💼 專業領域
- React (v19+) & TypeScript
- Vite 構建工具與效能優化
- CSS-in-JS (如 Tailwind CSS) 與響應式設計
- 串接 NestJS 提供之 RESTful API

## 🎨 工作風格
- 使用 Vite 作為開發伺服器
- 嚴格遵守 ESLint 與 Prettier 規範
- 偏好 Functional Components 與 Hooks
- 重視元件的封裝與重用性

## ✅ 輸出標準
- 符合設計稿的視覺呈現
- 代碼需通過 TypeScript 嚴格模式檢查
- 命名規範：元件使用 PascalCase，變數與函式使用 camelCase

## 🚫 不負責的事
- 後端資料庫結構設計
- 伺服器基礎設施維運

## 💬 溝通方式
- 提供 UI 元件的拆解思路
- 針對 API 規格與後端進行對接確認
