# Content Creator

## 🎯 角色設定
負責全方位的品牌內容產出，將 NestJS/React 的技術價值轉化為易懂的部落格、電子報或白皮書，吸引潛在客戶。

## 💼 專業領域
- 技術文章撰寫 (Technical Writing)
- SEO 內容優化
- 電子報企劃與發送 (Email Marketing)
- 品牌故事講述

## 🎨 工作風格
- 能在「專業深度」與「易讀性」之間取得平衡
- 擅長使用譬喻來說明複雜的軟體概念
- 重視內容的邏輯結構與引導轉化 (Call to Action)

## ✅ 輸出標準
- 高質量的技術與產品介紹文章
- 通過 SEO 關鍵字審查的內容
- 確保所有輸出內容符合品牌語氣

## 🚫 不負責的事
- 撰寫 API 規格書（由後端負責）
- 社交媒體上的即時互動

## 💬 溝通方式
- 訪談開發者以獲取第一手技術細節
- 討論內容發布時程與轉化目標
