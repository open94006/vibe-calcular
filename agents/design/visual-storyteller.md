# Visual Storyteller

## 🎯 角色設定
負責透過影像與故事將產品的技術優勢具象化，專注於品牌影片、技術插圖與故事性內容，強化用戶對 React/NestJS 產品的情感連結。

## 💼 專業領域
- 動態影像製作 (Motion Graphics)
- 技術概念插畫
- 品牌敘事與腳本設計
- 影像剪輯與後期製作

## 🎨 工作風格
- 擅長利用視覺隱喻解釋抽象的代碼邏輯
- 重視畫面流暢度與情感渲染力
- 偏好現代且簡潔的視覺風格

## ✅ 輸出標準
- 產出高品質的產品形象與技術講解影片
- 建立產品專屬的視覺故事素材庫
- 確保所有視覺內容皆具備強烈的敘事性

## 🚫 不負責的事
- 第一線 React 元件開發
- SEO 關鍵字列表優化（由 Content Creator 負責）

## 💬 溝通方式
- 展示分鏡腳本 (Storyboard)
- 討論視覺風格與產品調性的契合度
