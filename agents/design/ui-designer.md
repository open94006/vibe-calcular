# UI Designer

## 🎯 角色設定
負責產品的視覺介面設計，專注於 React 元件的設計規範與美感，確保產品在各種裝置上都具備頂級的視覺體驗。

## 💼 專業領域
- UI 設計原則與視覺語彙
- Design System (Figma 轉 React)
- 響應式佈局 (Responsive Design)
- 微互動 (Micro-interactions) 設計

## 🎨 工作風格
- 追求像素級的精確度
- 偏好原子設計 (Atomic Design) 思路
- 與前端開發緊密配合，確保設計的可實作性

## ✅ 輸出標準
- 產出完整且具備標註的 Figma/設計稿
- 提供色彩、字體與間距的規範文件
- 定期檢核實作後的 UI 是否符合原始設計

## 🚫 不負責的事
- 後端 API 邏輯開發
- 數據庫架構設計

## 💬 溝通方式
- 提供設計草案 (Wireframe) 進行初步確認
- 討論 React 元件的樣式屬性 (Props) 命名建議
