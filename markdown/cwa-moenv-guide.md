# 台灣開放資料 API Key 申請指南

為了讓天氣查詢功能優先使用台灣在地資料，建議您申請以下兩個服務的 API Key：

## 1. 中央氣象署 (CWA) - 天氣資料

中央氣象署提供最準確的台灣在地天氣觀測與預報資料。

1. **註冊帳號**：
   前往 [中央氣象署開放資料平臺](https://opendata.cwa.gov.tw/) 並註冊會員（可使用 Email）。

2. **取得授權碼 (API Key)**：
   登入後，點擊右上角的「會員資訊」或「API 授權碼」。您會看到一組授權碼，例如 `CWB-XXXXXXXX-XXXX-XXXX-XXXX-XXXXXXXXXXXX`。

3. **設定環境變數**：
   將此授權碼填入您的 `.env` 檔案中：
    ```env
    CWA_API_KEY=您的授權碼
    ```

## 2. 環境部 (MOENV) - 空氣品質資料

環境部提供全台測站的即時空氣品質指標 (AQI)。

1. **申請 API Key**：
   前往 [環境資料開放平臺](https://data.moenv.gov.tw/)。
   通常在 API 說明頁面或會員中心可以申請 API Key。如果您找不到直接申請的入口，可以使用平台提供的測試 Key 或尋找「申請 API 存取」的選項。

    _備註：若您暫時無法取得環境部 API Key，系統將會嘗試使用 OpenWeatherMap 的空氣品質資料作為備案，或者您可以稍後再補上。_

2. **設定環境變數**：
   取得 Key 後，填入 `.env` 檔案：
    ```env
    MOENV_API_KEY=您的APIKey
    ```

---

**設定完成後，請重新啟動後端伺服器以套用變更。**
